Introduction

Throughout this project, I crafted a dedicated website for the iconic television series Rick and Morty. All the engaging content showcased on the page is dynamically fetched from a third-party API.

Main Objectives

  • 🚀 Obtained information from third-party APIs
  • 🎨 Enhanced my proficiency in HTML, CSS, and JavaScript
  • 📘 Introduced TypeScript for enhanced code clarity
  • 🌐 Gained insights into seamless API integration with the client
  • ⚙️ Improved asynchronous data retrieval skills using AJAX
  • 🏗️ Structured the web project for clarity and order

General Analysis

My primary focus was to fashion a visually appealing website capable of presenting all facets of the series—episodes, characters, and locations. Users can effortlessly navigate through the dynamic content, showcasing the power of seamless information retrieval.
